# Brief Report

## Project Name
MyBindle Landing Page

## Tools Used
React (Vite), Tailwind CSS v4, Framer Motion, Lucide Icons, React Helmet Async.

## Time Taken
6 hours

## Tech Stack
Vite, React, Tailwind v4, Framer Motion. The Dark Mode was implemented using the new Tailwind v4 @tailwind strategy.

## Challenges & Solutions
- **Challenge:** Implementing Dark Mode in Tailwind v4.
  **Solution:** Used the new CSS-first @tailwind strategy instead of the old config file.

- **Challenge:** Pixel-perfect alignment of Figma's absolute positioning.
  **Solution:** Carefully translated pixel values to Tailwind's arbitrary value classes.

## Performance
Mention that lazy loading and WebP images were used for high Lighthouse scores.
